# Proxy TLS Legacy para el Ministerio

Este es un proxy Serverless diseñado para ejecutarse en Vercel. Permite hacer peticiones HTTP a servidores antiguos que devuelven el error `unsafe legacy renegotiation disabled`.

## Despliegue automático

Haz clic en el botón de abajo para desplegar tu propia copia de esta API en Vercel gratis en 1 minuto:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/nateRiver111/proxy-ministerio-tls)

## Uso
Una vez desplegado, usa la URL así:
`https://TU-URL-DE-VERCEL.vercel.app/api/proxy?url=URL_DEL_MINISTERIO_AQUI`
